公司概况

公司介绍： 公司于2015年4月经重大资产重组后由哈药集团三精制药股份有限公司更名而来，公司控股股东是哈药集团股份有限公司。公司现有哈药集团医药有限公司和哈药集团三精医院投资管理有限公司两家全资子公司，主营医药批发和零售业务，是国内知名的药品流通企业，黑龙江省医药商业的龙头企业。公司以先进的经营理念和专业化的优质服务，树立起“从商有德、诚信服务”的医药商业领导品牌，在全国医药商业的各项排名榜上，均处于领先地位。公司批发业务拥有健全、强大的销售网络，既覆盖了全省的终端医疗；建立了辐射全省的稳固、快速的商业分销渠道，具有很强的市场调拨能力，已拓展到吉林、内蒙古等省外市场。黑龙江省大型三级医院100%全覆盖，在黑龙江省内各三级甲等医院的销售额均列第一名，大中型批发企业和连锁药店100%覆盖。伴随基药的深入推广，省内基层医疗卫生机构覆盖率达60%以上。目前批发企业共有销售网点2800多家，具有强大的网络覆盖能力和市场掌控能力。

主要业务和产品： 主营业务:医药批发,医药零售等医药商业业务.

前十大股东信息：

人民同泰（600829.SH）前十大股东结构高度集中，哈药集团股份有限公司以74.8%的持股比例长期稳居第一大股东，且持股未发生变动，体现其绝对控股地位。其余股东持股比例均低于1%，呈现分散化特征。

截至2025年9月30日，除哈药集团外，其余前十大股东均为自然人或金融产品，无任何股东持股比例超过5%。因此，在剔除公募基金、私募基金等金融产品后，不存在需背景介绍的非公募大股东。历史数据中亦未出现其他持股超5%的非金融产品类股东。

近期主要股东变动方面：
- 周顺东在2025年三季度增持81,200股（持股比维持0.2%），为最新可查的明确增持行为；
- 王玲玲于2025年一季度及二季度分别增持13万、10.8万股，但2025年三季度持股未变；
- 机构股东层面，高盛、摩根士丹利、UBS等曾出现在2024年及此前报告期，但自2025年起已退出前十大名单；
- 多只私募基金（如金安产业鑫选）和QFII机构（如领航投资）曾在2023–2024年持有，但近期持股比例下降或退出前十。

综上，公司股权结构稳定，由哈药集团绝对控制，中小股东持股分散，近期无重大控股权变动，二级市场股东以短线交易型自然人为主，机构参与度显著降低。

公司历史沿革和创始人背景：

人民同泰（全称：哈药集团人民同泰科技股份有限公司）前身为哈尔滨市医药采购供应站，始建于1951年，是新中国成立初期国家建立的区域性医药流通骨干企业。公司于1996年在上海证券交易所上市（股票代码：600829），后经多次重组整合，2011年由哈药集团完成整体借壳上市，成为哈药集团旗下核心医药商业平台。

公司无传统意义上的“创始人”，实控人为哈尔滨市人民政府国有资产监督管理委员会，通过哈药集团间接控股，属国有控股企业。现任实际控制人与创始背景一致，均为地方政府主导的国有体系。公司核心管理层由国资委任命，职业经理人团队运营。

重要发展阶段包括：2011年借壳上市实现资本化；2015年深化分销网络建设，覆盖黑龙江全省；2020年以来聚焦“批零一体化”战略，强化零售连锁布局，旗下拥有人民同泰连锁药店超300家。近五年持续推进数字化转型与集采降本增效，在医保改革背景下巩固区域龙头地位。核心驱动力为国企资源支持、区域市场深耕及哈药集团工业协同优势。

现任总经理顾丛峰为哈药体系资深高管，硕士学历，高级工程师，长期任职于哈药系统，具备丰富的医药商业管理经验。独立董事韩东平、武滨等来自高校与行业权威机构，提升治理水平。

收入结构分析

收入结构和主要收入贡献来源：

根据人民同泰2020–2024年主营业务构成数据，结合医药流通行业特征及公开信息分析，其收入结构以医药批发为核心，零售与多元化产品为补充。以下从收入科目对应的产品/服务内容及客户群体、主要收入来源占比两个维度进行专业解析。

一、各收入科目的产品内容与客户群体

  1. 批发-医疗分销：该业务指公司向医疗机构（如公立医院、社区卫生服务中心、乡镇卫生院等）直接销售药品和医疗器械。产品主要包括处方药、中成药、基础抗生素、慢性病用药（如降压药、降糖药）等医保目录内主流药品。客户为具备资质的公立及民营医疗机构，合作关系稳定，订单量大且持续性强，是公司最核心的业务板块。

  2. 批发-商业调拨：指公司作为区域医药流通枢纽，向下游药品批发企业、连锁药店或单体药店进行药品调拨分销。产品涵盖非独家代理的普药、OTC药品及部分保健品。客户为区域性中小型医药经销商和零售终端，交易频次高但毛利率较低，体现公司在渠道网络中的集散功能。

  3. 中、西成药：此项为零售业务收入，主要来自公司自营药店（如“人民同泰”连锁药房）直接面向个人消费者销售中成药、化学制剂等OTC及处方药（凭方销售）。客户为终端患者及家庭用户，注重品牌、便利性与专业服务，毛利率高于批发业务。

  4. 医疗器械、计生用品及其他：包括家用医疗器械（如血压计、血糖仪）、医用耗材（注射器、口罩）、避孕药具等，通过零售门店或小批量批发实现销售，客户覆盖个人消费者与基层医疗机构。

  5. 保健品：主要为维生素、钙剂、蛋白粉等营养补充类产品，通过药店渠道销售，目标客户为中老年及亚健康人群，属高毛利品类但收入占比较低。

  6. 中药饮片：包括常见中药材切片（如黄芪、当归），主要供应医院及中医诊所，亦有少量零售，受中医诊疗需求影响较大。

  7. 其他主营业务与咨询费：可能涵盖物流服务、供应链管理咨询、药品信息化系统支持等增值服务，客户为合作医疗机构或商业伙伴，收入规模小但具协同效应。

  8. 出租固定资产：源于自有仓储设施或物业的租赁收入，属非核心业务。

二、按产品划分的主要收入来源占比分析

以最新报告期2024年数据为基础，人民同泰的收入高度集中于医药批发领域，合计占比达81.8%（医疗分销59.3% + 商业调拨22.5%），其中“批发-医疗分销”连续多年稳居首位，尽管2024年收入较2023年略有下降（61.5→59.6亿元），但仍保持主导地位。

零售端的“中、西成药”收入持续增长（2020年11.3亿 → 2024年16.2亿），占比提升至16.1%，反映其零售网络扩张与终端服务能力增强。相比之下，保健品、医疗器械等多元化产品合计贡献不足1%，尚未形成显著增量。

值得注意的是，医疗分销收入占比呈缓慢下降趋势（2020年62.2% → 2024年59.3%），而商业调拨与零售业务占比上升，表明公司正优化客户结构，拓展渠道覆盖广度，降低对单一医院客户的依赖。

综上，人民同泰的核心收入来源于面向医疗机构的药品批发，辅以广泛的商业调拨网络与不断壮大的自营零售体系。未来增长动能或将更多来自零售终端渗透与高毛利品类（如保健品、器械）的结构性提升，但当前仍是一家典型的区域性医药流通服务商，收入结构稳健但创新业务贡献有限。

客户构成和销售模式

人民同泰（原哈药集团人民同泰股份有限公司）作为区域性医药流通与零售龙头企业，主要依托黑龙江省市场，其业务涵盖药品批发、零售连锁及医疗健康服务。以下从下游客户构成与销售模式、上游供应链两个维度进行分析。

一、下游主要客户类型和构成

人民同泰的下游客户以B端医疗机构与C端零售消费者为主，形成“批零一体化”的双轮驱动模式。

  1. 医疗机构客户：包括公立医院、基层卫生院及社区诊所等。公司作为哈药集团体系内的核心分销平台，长期承担哈药产品在黑龙江地区的医院配送任务，与省内多家二级以上公立医院建立稳定供货关系。通过参与省级药品集中采购，成为中标品种的区域配送商，强化了对公立医疗渠道的渗透。此外，借助冷链物流能力，覆盖疫苗、生物制剂等高值专科药品的院内配送。

  2. 零售终端客户：以自有连锁药店为核心，同时覆盖其他社会药房。人民同泰运营“人民同泰”品牌连锁药店超300家，主要集中于哈尔滨及黑龙江地级市核心商圈，直面C端消费者。其零售板块受益于处方外流趋势，承接大量慢病用药、OTC药品及保健品销售。疫情期间，退热、止咳、抗病毒类药品需求激增，进一步提升了零售端营收占比。

  3. 第三方商业客户：包括其他医药流通企业、民营医院及互联网医疗平台。公司利用区域仓储与物流网络，为中小型经销商提供集采与代配服务，增强渠道控制力。同时，与京东健康、阿里健康等平台合作，开展O2O即时配送业务，拓展线上流量入口。

销售模式方面，公司采用“自营+代理+电商协同”的复合模式。对重点医院实行专职代表对接与学术推广；零售端推行会员管理、慢病跟踪等增值服务提升复购率；线上则通过美团、饿了么及自建小程序实现全渠道销售。2023年数据显示，批发业务占营收约60%，零售占比接近40%，体现出较强的渠道均衡性。

二、产业链上游基本情况与核心物料

人民同泰作为流通环节企业，不直接从事药品生产，其上游主要为药品生产企业与原料供应商，具有明显的“多源采购、品牌导向”特征。

  1. 核心供货方：以哈药集团内部企业为第一大供应源。哈药股份、哈药中药二厂、三精制药等子公司为其提供抗生素、心脑血管药、中成药等主力品种，保障货源稳定与价格优势。据公开财报，关联交易占比曾达30%以上，体现集团协同效应。

  2. 外部制药企业：广泛对接国内大型药企，如华润医药、石药集团、扬子江药业等，引入非竞争性品类补充产品线。在集采背景下，公司积极拓展中标品种代理权，增强对独家或专利药品的引进能力。

  3. 原料与制剂来源：虽不涉及原料药生产，但对上游原料供应波动敏感。核心物料包括化学原料药（如阿莫西林、头孢类）、中药材（用于中成药配制）及生物制剂原液。近年来受环保政策与国际供应链影响，部分原料药价格波动较大，公司通过集中采购、长单锁定等方式降低风险。

  4. 供应链管理：依托哈尔滨现代化物流中心，具备常温、阴凉、冷藏立体仓储能力，支持GSP合规配送。上游采购高度依赖信息化系统对接，实现订单自动化处理与库存动态预警，提升周转效率。

综上，人民同泰凭借区域渠道深耕与批零协同优势，构建起以医疗机构、零售终端为主体的多元化客户体系，并依托哈药集团资源与外部厂商合作，保障上游供给稳定性。未来在医保控费与行业集中度提升背景下，其区域龙头地位有望持续巩固。
