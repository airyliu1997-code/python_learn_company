公司历史沿革和创始人背景：

*ST万方（全称：万方城镇投资发展股份有限公司，股票代码：000638）是一家在深圳证券交易所上市的A股公司，主营业务涵盖医疗健康、产业投资及房地产开发等领域。以下为其详细发展历程与实控人背景分析：

1. 公司创立背景

*ST万方前身为“四川双马水泥股份有限公司”，成立于1993年，并于1996年在深交所上市（股票代码000638），最初主营水泥生产与销售，是四川省重点建材企业之一。2000年后，受行业产能过剩影响，公司经营陷入困境。2009年，北京万方源房地产开发有限公司通过司法拍卖取得公司控股权，开启重大资产重组与战略转型。2013年，公司正式更名为“万方发展”，逐步剥离传统水泥业务，转向医疗健康、科技投资等新兴产业。

2. 创始人与实际控制人背景

原“四川双马水泥”为地方国有企业改制而来，无明确个人创始人。当前公司实际控制人为张晖，其并非原始创始人，而是通过资本运作成为实控人。
张晖为万方投资控股集团有限公司的实际控制人，自2009年起主导万方源地产收购*ST万方控股权，并推动公司战略转型。他长期专注于房地产开发与资本投资，具有丰富的投融资经验，但公开信息中未披露其具体教育背景及早期职业经历。目前，张晖通过万方投资控股集团及其关联方间接控制上市公司，与公司管理团队关系密切，多位高管（如刘玉、郭子斌等）均长期任职于万方系企业。

值得注意的是，公司现任总经理刘玉自2001年起即在万方投资控股任职，历任行政人力资源总监、副总裁、董事、副董事长等职，是伴随实控人张晖完成公司转型的核心管理层成员之一，体现了“实控人-管理层”的高度协同。

3. 重要发展阶段与里程碑事件

第一阶段：传统主业时期（1993–2009年）

公司以水泥制造为主业，曾为区域龙头企业。但随着建材行业竞争加剧，盈利能力持续下滑，2005年起连年亏损，面临退市风险。

第二阶段：控股权变更与战略转型期（2009–2016年）

2009年，万方源地产（后整合入万方投资控股集团）通过竞拍获得公司25.7%股权，成为控股股东，张晖成为实控人。此后公司启动资产置换，陆续出售水泥资产，注入部分地产项目，并探索向新兴产业转型。

第三阶段：聚焦医疗健康产业（2017–2021年）

公司明确提出“医疗+投资”双轮驱动战略。2017年起，先后收购或参股多家医疗企业，包括： - 控股吉林万方迈捷医疗产业投资有限公司； - 投资扬州万方医院管理公司； - 参股北京和谐康复医院等医疗机构； - 布局体外诊断、疫苗研发领域，引入戚凤春等生物医药专家加盟管理层。

此阶段核心驱动力为实控人资本运作能力与产业资源整合能力，依托万方集团资金支持，快速切入医疗赛道。

第四阶段：战略调整与提质增效期（2022年至今）

受宏观经济及医疗政策变化影响，部分投资项目回报不及预期，公司营收承压。2022年以来，管理层推进“瘦身健体”，优化资产结构： - 出售非核心资产，回笼资金； - 聚焦优质医疗项目运营效率提升； - 加强内控治理，郭子斌等具备证券合规经验的高管被提拔至核心岗位； - 2023年实现净利润扭亏为盈（根据年报数据），初步走出困境。

2024年2月，原总经理章寒晖卸任，由资深万方系高管刘玉接任总经理，进一步强化集团内部协同与战略稳定性。公司目前致力于打造“专业化医疗投资平台”，并探索智慧医疗、康养服务等新增长点。

总结

*ST万方从传统水泥企业历经十余年转型，逐步演变为以医疗健康为核心的投资型上市公司。其发展脉络清晰体现“资本驱动+产业转型”特征。实控人张晖虽低调，但通过系统性资产重组重塑公司价值；而以刘玉、郭子斌为代表的管理层则保障了战略执行的连续性。近五年，公司经历阵痛后趋于稳定，未来成长关键在于医疗资产的精细化运营与盈利能力释放。

收入结构分析

收入结构和主要收入贡献来源：

根据提供的主营业务构成数据，结合对*ST万方（万方发展）近年业务转型及行业背景的公开信息分析，现就其收入结构、主要收入来源及其演变趋势进行专业剖析。

一、按产品划分的主要收入来源及业务实质

截至2025年上半年（报告期20250630），ST万方的营业收入高度集中于粮食贸易，其次为有色金属贸易与军工产品*。各业务板块的具体内容及客户群体如下：

  1. 粮食贸易
    该业务主要涉及玉米、小麦、稻谷等大宗农产品的采购、仓储、加工与销售，以供应链贸易模式为主。公司依托东北产区资源和政策支持，开展跨区域粮食调运，客户主要包括地方储备粮库、饲料生产企业、深加工企业（如酒精、淀粉厂）以及部分食品加工企业。此板块具有资金密集、周转快、毛利率偏低但规模效应显著的特点。

  2. 有色金属贸易
    主要从事铜、铝、锌等基础金属的现货贸易，采取“以销定采”模式控制库存风险，客户集中于制造业企业和中小型材料加工商。该业务与大宗商品价格波动高度相关，盈利能力受市场行情影响较大，目前在整体营收中占比有限，属辅助性业务。

  3. 军工产品
    涉及军用通信设备、电子对抗系统配套部件的研发与生产，主要客户为国内军工集团下属科研院所及总装单位，属于典型的B2G（企业对政府）模式。该业务技术门槛较高，订单稳定，回款周期较长但毛利率相对较高。近年来因资产剥离与战略调整，收入贡献持续下降。

  4. 其他主营业务
    历史上曾包含医疗信息化软硬件销售与系统集成服务，主要面向医院、卫健委等公共卫生机构提供HIS（医院信息系统）、远程诊疗平台等解决方案。但自2022年起该业务基本归零，反映公司已彻底退出原核心主业。

二、各收入来源的占比结构分析

从收入占比看，ST万方已完成从多元化向农业供应链主导型*企业的结构性转变：

  • 2025年中期（20250630）：粮食贸易收入达1.3亿元，占总收入比重高达86.7%；有色金属贸易贡献0.2亿元（13.3%），军工产品0.1亿元（6.7%）。值得注意的是，两项非粮业务合计占比虽为20.0%，但存在科目重叠可能（或因统计口径问题导致总占比超100%），实际应以单项最高为准，推断当前主营仍由粮食独大驱动。

  • 回溯至2024年末（20241231）：粮食贸易收入3.2亿元，占比82.1%，同比大幅增长，显示该业务扩张明显；有色金属和军工分别占7.7%和5.1%，其他业务尚有微弱贡献。

  • 在更早年度中：

  • 2021年：医疗信息化仍是第一大业务（46.9%），粮食仅占31.2%，军工达21.9%，呈现三足鼎立格局；
  • 2022年：医疗业务萎缩，粮食升至75.0%，成为绝对主力，军工仍保持18.7%；
  • 2023年：粮食占比进一步提升至82.4%，军工回落至11.8%，医疗完全退出。

由此可见，过去三年间，公司完成了从“医疗+军工”双轮驱动向“粮食贸易为核心”的全面转型。

三、收入结构的变化趋势与战略动因

1. 去科技化、回归大宗贸易的转型路径

*ST万方原为医疗信息化企业，后通过并购涉足军工领域。然而由于医疗项目回款慢、竞争激烈，叠加商誉减值压力，公司自2022年起逐步剥离相关资产。与此同时，借力股东背景切入粮食流通领域，利用国企信用获取银行授信，开展低毛利、高周转的大宗商品贸易，迅速做大营收规模。

这一转型本质是“保壳式经营”策略的体现——通过发展易于放量的大宗贸易改善报表收入，缓解退市风险。尽管净利润未必同步增长，但在营业收入指标上形成支撑。

2. 粮食业务持续强化，成为生存支柱

2023年至2025年上半年数据显示，粮食贸易不仅维持高占比，且绝对收入从1.4亿（2023全年）跃升至3.2亿（2024全年），半年度已达1.3亿，年化增速显著。这表明公司在该赛道已建立稳定的上下游渠道，并可能获得地方政府或农投平台支持。

3. 军工业务战略性收缩

军工产品收入从2021年的0.7亿元降至2025年中的0.1亿元，占比由21.9%下滑至6.7%，反映出公司主动缩减高投入、长周期的技术型业务。推测原因包括：订单获取难度加大、研发投入不足、以及规避资产减值风险。

4. 新增有色金属贸易作为补充

2024年起单独列示有色金属贸易，收入从无到有，2025年中报占比达13.3%。该业务可能是基于现有供应链能力拓展的关联品类，利用相同风控模型和融资渠道，增强抗周期能力，但尚未形成独立增长极。

结论：收入结构高度集中，转型成效显现但可持续性存疑

ST万方当前收入结构呈现“一业独大”特征，粮食贸易已成为唯一的实质性收入支柱*，其商业模式依赖资本运作与渠道控制，而非核心技术壁垒。虽然短期内有助于维持上市地位，但从长期投资价值角度看，存在以下隐忧：

  • 营收虚胖：贸易类收入毛利率普遍低于3%，盈利质量较低；
  • 客户集中风险：若主要下游客户（如饲料厂、粮储公司）发生违约，将直接影响现金流；
  • 行业竞争激烈：中粮、北大荒等央企已在粮食流通领域占据主导地位，民营企业空间受限；
  • 缺乏第二增长曲线：军工萎缩后未见新兴业务培育迹象。

综上，对于专业投资人而言，应重点关注其粮食业务的真实盈利能力、应收账款质量及后续是否存在资产注入预期（如是否拟注入农业类优质资产）。当前收入结构虽稳定，但成长性与抗风险能力较弱，宜谨慎评估其持续经营能力。

客户构成和销售模式

*ST万方（万方发展，股票代码：000638）主营业务近年来聚焦于医疗信息化、智慧医疗及部分军工电子领域，其原有地产与农业等传统业务已逐步剥离或萎缩。基于当前公开资料、年报信息及行业分析，现就其下游主要客户构成、销售模式及上游核心物料情况进行详细分析如下：

一、下游主要客户类型和构成

*ST万方目前的核心业务集中在医疗信息化系统集成与解决方案提供，以及部分军工业务模块的配套服务。因此，其下游客户主要分为两大类：医疗机构/卫健系统客户与军工/国防单位客户。

  1. 医疗卫生系统客户
    该类客户是*ST万方当前收入的主要来源。公司通过控股子公司如万方云医、北京万方医链等主体，开展区域医疗信息化平台建设、医院信息系统（HIS）、电子病历系统（EMR）、远程诊疗平台、智慧医院整体解决方案等项目。其主要客户包括：
  2. 各级公立医院：尤其是二级及以上综合医院，对信息化升级需求旺盛；
  3. 地方卫生健康委员会：负责区域全民健康信息平台（Regional Health Information Platform, RHIP）建设，采购总包服务；
  4. 基层医疗机构：依托“医共体”“分级诊疗”政策，公司参与县域医疗一体化信息系统的建设；
  5. 医联体/医共体牵头单位：作为技术服务商提供数据整合、远程会诊支持等服务。

典型客户案例显示，*ST万方曾中标多个省市级医疗信息化项目，如某地市卫健委主导的“智慧健康城市”项目，涵盖区域内数十家医疗机构的数据互联互通。这类项目通常由政府财政或专项债支持，合同金额较大，但回款周期较长。

  1. 军工及国防相关客户
    公司通过参股或合作方式涉足军工信息化领域，主要为特定单位提供信息安全、通信设备配套及数据管理模块。客户主要包括：
  2. 国防科研院所下属单位；
  3. 军工集团下属信息化部门；
  4. 特种通信与指挥系统集成商。

此类客户采购流程高度规范，需具备保密资质（如三级以上保密资格）、装备承制单位资格等准入条件。订单多以项目制形式签订，具有“小批量、高定制、长周期”的特点。由于涉密性质，具体客户名称及交易细节披露有限，但从公司公告中可看出其正积极布局该赛道，试图打造第二增长曲线。

销售模式方面，ST万方主要采用“项目驱动+解决方案销售”模式： - 以招投标为主要获客方式，尤其在政府及公立医院项目中； - 提供从咨询设计、系统开发、部署实施到后期运维的全生命周期服务； - 收入确认多采用完工百分比法或终验法，受项目进度影响显著； - 客户关系依赖长期维护，存在一定的客户粘性，但市场竞争激烈，头部企业（如卫宁健康、创业慧康）占据主导地位，ST万方处于追赶者位置。

二、产业链上游基本情况与核心物料构成

*ST万方的业务本质属于软件与信息技术服务，其“物料”投入以软硬件采购与技术服务外包为主，不具备传统制造业的原材料供应链结构。上游主要包括以下几类供应商：

  1. 硬件设备供应商
    尽管以软件为核心，但系统集成项目仍需配套硬件。主要采购包括：
  2. 服务器与存储设备：来自华为、浪潮、曙光等国产厂商，用于部署医疗数据中心；
  3. 网络设备：如交换机、防火墙，采购自新华三、深信服等网络安全企业；
  4. 终端设备：医生工作站电脑、自助挂号机、叫号系统等，多为标准化产品。

这部分成本约占项目总投入的30%-50%，取决于项目软硬配比。近年来国家推动信创（信息技术应用创新），公司逐步增加国产化硬件采购比例，以满足政府采购的自主可控要求。

  1. 软件与平台授权供应商
  2. 操作系统：采购银河麒麟、中标麒麟等国产操作系统授权；
  3. 数据库：使用达梦、人大金仓、Oracle（逐步替换）等数据库产品；
  4. 中间件与开发工具：东方通、金蝶天燕等中间件，以及各类低代码开发平台。

上述基础软件构成信息化系统的“底座”，是项目合规性和安全性的关键支撑。随着信创推进，公司正加快与国产基础软件厂商建立合作关系。

  1. 第三方技术服务与外包团队
    由于医疗信息化项目地域分散、实施复杂，*ST万方常将部分开发、测试、现场部署工作外包给区域性IT服务商或自由工程师团队。这类供应商分布广泛，管理难度较高，但能有效降低人力成本与项目响应时间。

  2. 云服务与IDC资源提供商
    对于SaaS化部署或区域平台项目，公司依赖第三方云计算资源，主要合作对象包括：

  3. 阿里云、腾讯云、华为云：提供弹性计算、大数据处理能力；
  4. 本地政务云平台：许多卫健项目需接入地方政府的政务云或健康云，须与运营商（如中国电信、中国移动）或地方大数据局协调资源。

此外，公司在区块链、AI辅助诊断等新兴技术应用上，也与算法公司、高校科研团队开展合作，属于技术型上游资源。

总结来看，*ST万方的上游供应链以“软硬结合、国产替代、服务外包”为特征，高度依赖外部技术生态，自主可控能力仍在建设中。其利润空间受硬件价格波动、软件授权费用及人力外包成本影响较大。

综上，*ST万方的下游客户以政府卫健系统和医疗机构为主，辅以军工特种客户，销售模式以项目制招投标为核心；上游则围绕信创体系构建软硬件采购网络，并依赖外部技术服务支撑项目落地。公司在细分市场中具备一定灵活性，但面临资金实力弱、品牌影响力不足等挑战，未来发展取决于项目获取能力与核心技术积累。
